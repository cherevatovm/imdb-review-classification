# Review Classifier
Веб-приложение на Django, которое с помощью модели машинного обучения классифицирует текстовые отзывы о фильмах как позитивные или негативные и вычисляет приблизительный рейтинг (1-10).
Модель обучена на датасете IMDB Movie Reviews, используется TF-IDF + SGDClassifier.

## Предобработка текста
Перед векторизацией текст проходит несколько этапов обработки:
1. Удаление HTML-тегов
2. Нормализация текста
    - перевод в нижний регистр
    - удаление небуквенных символов
3. Обработка эмодзи
4. Удаление стоп-слов
5. Стемминг с помощью SnowballStemmer

После обработки текст преобразуется в TF-IDF вектор.

## Сравнение моделей
Для задачи классификации были протестированы несколько алгоритмов из `scikit-learn`.
Подбор гиперпараметров проводился с помощью GridSearchCV, где `cv=5`.

| Модель | Accuracy (CV) | Accuracy (Test) |
|------|------|------|
| Logistic Regression | 0.894 | 0.876 |
| SGDClassifier | **0.894** | **0.881** |
| Random Forest | 0.857 | 0.857 |
| Multinomial Naive Bayes | 0.862 | 0.816 |

Лучшие результаты показали Logistic Regression и SGDClassifier.
В качестве итоговой модели был выбран SGDClassifier, поскольку он:
- показал наибольшую точность на тестовой выборке
- хорошо работает с разреженными TF-IDF признаками
- быстрее обучается и лучше масштабируется на больших текстовых данных

Итоговые параметры SGDClassifier:
```
alpha = 1e-05
penalty = L2
loss = log_loss
```

Модель и TF-IDF векторизатор были сохранены с помощью `pickle` и используются в Django-приложении.
